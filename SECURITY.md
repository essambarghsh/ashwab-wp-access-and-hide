# Security Policy

## Supported Versions

We support the latest version of the plugin. Please ensure you are using the most recent release to receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please **DO NOT** open a public issue. Instead, please report it responsibly:

1.  Email us at [dev@ashwab.com](mailto:dev@ashwab.com) (or the contact email provided in the plugin).
2.  Provide a detailed description of the vulnerability and steps to reproduce it.
3.  We will acknowledge your report and work to address the issue as quickly as possible.

We appreciate your help in keeping our users safe!
